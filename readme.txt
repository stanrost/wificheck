The application is about geofencing.

- You can add, view, update and delete new geofences
- The action when you leave or enter is wifi related
- The wifi will turn on or off and a notification can be generated 

Libraries used within the application:
- implementation 'com.karumi:dexter:5.0.0'
- implementation 'com.google.android.gms:play-services-location:16.0.0'
- implementation 'com.google.android.gms:play-services-maps:16.1.0'
